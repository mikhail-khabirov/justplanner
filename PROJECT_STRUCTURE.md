# JustPlanner — Структура проекта

## Описание

**JustPlanner** — это веб-приложение для планирования задач, построенное на React + TypeScript + Vite. Приложение предоставляет календарный вид задач с возможностью drag-and-drop, поддержкой подзадач, цветовой маркировкой и интеграцией с Gemini API.

---

## Описание компонентов

### Главные файлы

#### `App.tsx` (~480 строк)
Главный компонент приложения, содержит:
- **State Management** — управление списком задач и колонок
- **Навигация** — переключение между неделями (`handlePrevWeek`, `handleNextWeek`)
- **Drag & Drop** — перемещение задач между колонками (`moveTask`)
- **CRUD операции** — создание, обновление, удаление задач
- **Статистика** — подсчёт выполненных задач
- **Категории** — Входящие, Срочно, Когда-нибудь, Идеи

#### `types.ts`
Определяет структуры данных:
- `TaskColor` (enum) — цвета задач (default, yellow, green, purple, red, blue)
- `Subtask` — подзадача (id, content, completed)
- `Task` — задача (id, content, columnId, hour, color, completed, subtasks)
- `Column` — колонка календаря (id, dateLabel, dayLabel, isToday, isWeekend, isHoliday)

#### `utils.ts`
Утилитарные функции:
- `generateId()` — генерация уникальных ID
- `formatDate()` / `getDayName()` — форматирование дат на русском
- `toISODate()` — преобразование в ISO формат
- `generateColumns()` — генерация колонок календаря на заданный период
- `HOLIDAYS_2026` — список праздничных дней РФ

### React Компоненты

#### `Column.tsx` (~262 строки)
Колонка календаря с временными слотами:
- Отображение задач по часам (5:00–23:00, 00:00)
- Поддержка drag & drop
- Индикация выходных и праздников
- Сворачиваемые ранние часы (5-7)

#### `TaskItem.tsx` (~136 строк)
Карточка задачи:
- Цветовая маркировка
- Checkbox для отметки выполнения
- Drag & drop перетаскивание
- Индикатор подзадач (n/m выполнено)

#### `TaskModal.tsx` (~379 строк)
Модальное окно редактирования задачи:
- Редактирование названия
- Выбор даты (календарь)
- Выбор времени
- Выбор цвета
- Управление подзадачами (добавление, отметка, удаление)
- Удаление задачи

#### `QuickAddInput.tsx` (~126 строк)
Компонент быстрого добавления задачи:
- Текстовое поле с автофокусом
- Выбор цвета
- Подтверждение по Enter, отмена по Escape
- Сохранение при клике вне компонента

## Особенности

1. **Локализация** — интерфейс на русском языке
2. **Праздники РФ** — отмечены в календаре (2026 год)
3. **Демо-дата** — "сегодня" зафиксировано как 30 января 2026
4. **AI Studio** — интеграция с Google AI Studio через Gemini API
5. **Современный дизайн** — тени, анимации, цветовое кодирование
